HR Data Analytics Dashboard | Power BI

This project presents an end-to-end HR Data Analytics Dashboard built using Power BI, designed to help HR teams monitor workforce performance, identify risk areas, and make data-driven decisions.

Project Overview: 

The dashboard analyzes key HR metrics including employee demographics, attendance, performance ratings, hiring trends, and attrition patterns. It enables HR teams to quickly assess employee distribution, identify high-risk departments, and understand factors influencing employee turnover.

Key Objectives: 

* Provide a unified view of employee data across age, department, gender, and job roles.

* Identify trends in attrition, retention, and hiring patterns.

* Highlight workforce gaps and productivity patterns.

* Support HR leaders in strategic planning and employee engagement.

DASHBOARD FEATURES

Employee Demographics: 

* Age group & gender-wise distribution

* Department and job role segmentation

* Average salary insights

* Hiring & Workforce Summary: 

* Total employees

* New hires vs. exits

* Department-wise headcount trends

Attrition Analysis: 

* Overall attrition rate

* High-attrition departments

* Breakdown by age, role, salary slab, and job satisfaction

Performance Insights: 

* Performance rating distribution

* Top vs. bottom-performing departments

Tools & Technologies: 

* Power BI (Data modeling, DAX measures, interactive dashboard)

* Excel / CSV (Data cleaning & preprocessing)

* Power Query (ETL transformations)

Repository Contains: 

* .pbix file of the HR dashboard

* Dataset (CSV/Excel)

* Screenshots of dashboard pages

* README with project explanation

Impact: 

Helps HR managers quickly identify workforce patterns, reduce attrition risks, improve performance monitoring, and make informed HR decisions.
